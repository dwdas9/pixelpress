# Session-Start Protocol

The onboarding path, for anyone — human or model. This is a
continuation, not a fresh audit.

1. Read `CURRENT_STATE.md` in full.
2. Read only the files in its Context Dependency Index — nothing
   implied around them, nothing "probably relevant."

Do not scan, grep, or walk the rest of the repository. Do not open any
other documentation unless `CURRENT_STATE.md` names it explicitly for
this task.

If an indexed file is missing, or contradicts what `CURRENT_STATE.md`
claims about it: stop and report this. Do not fall back to exploring
the repository to compensate — a broken index is a bug in the previous
checkpoint, not a license to widen scope.

Then report back: the current milestone, the next task, the exact
files ingested, and any contradiction found (or "none"). Proceed to
the next task only after that.
